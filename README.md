# A template to bootstrap Wealize Frontend projects

## Template info, delete after use the template

To start a new project, click on the "Use this template" button.

## Installation

```bash
yarn install
```

## Run in development

### Run Next.js app

```bash
yarn dev
```

### Run Jest tests

```bash
yarn jest
```

### Run Cypress end-to-end tests

```bash
yarn cypress
```
